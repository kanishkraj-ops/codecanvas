Mondrian Style Art

Mondrian-style abstract art using only pure HTML and CSS, drawing inspiration from Piet Mondrian's iconic geometric compositions. This project showcases the power of modern CSS — using grid layout, precise positioning, and color theory to mimic the bold lines and primary color blocks characteristic of neoplasticism. No images, no JavaScript — just code turning design into visual expression.

Port Scanner

I developed a Python-based port scanner that efficiently scans a target host for open ports, helping identify potential network vulnerabilities. The tool utilizes socket programming and multithreading to speed up the scanning process while maintaining accuracy. It’s a great starting point for understanding how network reconnaissance works in real-world penetration testing and cybersecurity assessments.
